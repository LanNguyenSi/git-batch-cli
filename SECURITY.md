# Security Policy

## Supported Versions

Security fixes are applied to the latest released version.

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive problems.

Instead, report:

- the affected command or code path
- a minimal reproduction
- the potential impact
- any suggested remediation

Until a dedicated private contact is configured for this project, use a private communication channel with the maintainer rather than a public issue tracker.
